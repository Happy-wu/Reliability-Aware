# Legacy Synthetic Experiments

These scripts reproduce the earlier synthetic reliability and Q/K experiments.
They are retained for provenance but are no longer the main research pipeline.

Run them from the project root, for example:

```bash
python scripts/legacy_synthetic/run_batch.py --help
python scripts/legacy_synthetic/run_reliability_suite.py --help
python scripts/legacy_synthetic/run_synthetic_finalists.py --help
```

The current main experiment entry point is:

```bash
python run_expert_fusion_suite.py --help
```
